# Avitation Launchpad

Welcome to the Avitation Launchpad repository.

## Overview
This project serves as a launchpad for aviation-related resources and tools.

## Getting Started

### Prerequisites
* Node.js
* npm

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Scripts
* `npm start` or `npm run dev`: Runs the app in development mode.
* `npm run build`: Builds the app for production.
